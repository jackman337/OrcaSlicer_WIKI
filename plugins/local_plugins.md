# Local Plugins

Local plugins are plugin files you install manually from your computer.

## Install a Local Plugin

1. Open **File** > **Plugins**.
2. Click the arrow next to **Browse plugins**.
3. Click **Install local plugin**.
4. Select the plugin file.

![install-local-plugin](https://github.com/OrcaSlicer/OrcaSlicer_WIKI/blob/main/images/plugins/install-local-plugin.png?raw=true)

After installation, the plugin appears in the list. A newly installed plugin may be **Inactive** until you enable it.

![local-plugin-installed-inactive](https://github.com/OrcaSlicer/OrcaSlicer_WIKI/blob/main/images/plugins/local-plugin-installed-inactive.png?raw=true)

To activate the plugin, check the box in the **Activate** column. Expand the plugin row to see the features provided by that plugin. Each feature can also have its own checkbox.

![local-plugin-activated-capabilities](https://github.com/OrcaSlicer/OrcaSlicer_WIKI/blob/main/images/plugins/local-plugin-activated-capabilities.png?raw=true)
