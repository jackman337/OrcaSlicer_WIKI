# Option Modes

OrcaSlicer offers four levels of configuration options.  
These levels are designed to hide complexity from users who do not need it, while still providing access to advanced settings for power users.  
Each level builds upon the previous one, adding more options and features as you progress through them.

![modes_switch](https://github.com/OrcaSlicer/OrcaSlicer_WIKI/blob/main/images/GUI/modes_switch.gif?raw=true)

- **Simple**: Only the essential options for slicing. Ideal for beginners or those who want a quick setup.
- **Advanced**: Suitable for users who have some experience with slicing and want to fine-tune their settings.
- **Expert**: Some experimental and those that require a deep understanding of the 3D printing process.
- **Developer**: The highest level of configuration, providing access to all available options. This level is intended for developers or users who want to experiment with every aspect of the Slicer. Some options in this mode may be unstable or not fully tested, so it is recommended to use this mode with caution. **To enable Developer mode, you need to open OrcaSlicer Preferences/Developer/Settings/Developer Mode**.

> NEW FEATURE: **Expert Mode**  
> Available in: [Nightly builds](https://github.com/OrcaSlicer/OrcaSlicer/releases/tag/nightly-builds) or Releases greater than **2.3.2**.

> [!NOTE]
> When using Developer mode, the Mode selector will be disabled.  
> This is because Developer mode provides access to all options, and there is no need to switch between modes.
